# Slive_prj  
  
KSL 수어 통역 시스템 (Workspace)
이 프로젝트는 Flask를 기반으로 한 KSL(한국 수어) 통역 및 제스처 학습 시스템의 백엔드 워크스페이스입니다. Google의 Gemini API를 활용하여 수어 데이터를 처리하고 모델링할 수 있는 구조를 갖추고 있습니다.

✨ 주요 기능
통합 워크스페이스: 수어 데이터를 수집하고 학습 모델의 정보를 관리하는 대시보드를 제공합니다.

Gemini AI 통합: Google Generative AI를 사용하여 복잡한 수어 패턴 분석 및 통역 로직을 지원합니다.

제스처 샘플링 API: 저장된 수어 랜드마크 데이터를 기반으로 랜덤 샘플을 추출하여 시각화 및 학습에 활용합니다.

RESTful API: 프론트엔드와 통신하여 모델 정보 및 제스처 데이터를 효율적으로 주고받습니다.

📂 파일 구조
app_flask.py: Flask 서버 메인 로직 및 API 엔드포인트 정의

requirements.txt: 프로젝트 실행을 위한 Python 라이브러리 목록

data/: 수어 제스처 데이터(JSON) 저장 폴더 (자동 생성)

trained-model/: 학습된 모델 파일 저장 폴더 (자동 생성)

.env: API 키 등 환경 변수 설정 (보안 주의)
