# Slive_prj  
  
✨ 핵심 기능 (Key Features)
🖥️ 통합 워크스페이스 대시보드

수어 제스처 데이터 수집 현황을 한눈에 파악합니다.

학습된 AI 모델의 정보와 상태를 효율적으로 관리합니다.

##🤖 Google Gemini AI 통합

google-generativeai를 연동하여 고도화된 수어 패턴 분석을 수행합니다.

LLM 기반의 자연스러운 수어-언어 번역 로직을 지원합니다.

##📊 제스처 샘플링 API

data/ 폴더 내의 랜드마크 JSON 데이터를 분석합니다.

학습용 데이터 검토를 위해 랜덤 샘플을 시각화 데이터로 제공합니다.

##⚡ RESTful API 통신

프론트엔드와 유연하게 통신하여 실시간 제스처 데이터 및 모델 파라미터를 교환합니다.
